⚠️ Configuración requerida
Para guardar datos desde el ERP directamente a Google Sheets sin pasar por el formulario de Google, el Apps Script debe estar desplegado como Web App.

Pasos:
1. Haz click en "Abrir Apps Script" abajo
2. Implementar → Nueva implementación
3. Tipo: Aplicación web
4. Ejecutar como: jrojas1959@gmail.com
5. Quién tiene acceso: Cualquiera
6. Implementar → Copiar la URL generada
7. Pegar la URL en Base de datos → Configuración Web App
